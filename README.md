# Employee Management System SQL Project

## Overview

This project demonstrates SQL fundamentals through the design and implementation of an Employee Management System database.

The project covers database creation, table relationships, data insertion, and analytical SQL queries.

## Database Schema

The project consists of four tables:

* departments
* employees
* projects
* former_employees

### Relationships

* One department can have many employees.
* One employee can be assigned to a project.
* Employees may report to another employee (manager relationship).
* Former employees are maintained separately for UNION operations.

* INNER JOIN

* GROUP BY

* Subqueries
* Views

* UPDATE
* DELETE

## Sample Business Questions

1. Show employee and department information.
2. Find employees earning above average salary.
3. Display employee-manager relationships.
4. Calculate department salary statistics.
5. Identify departments with more than two employees.

## Key Learning Outcomes

* Relational database design
* Foreign key constraints
* Data integrity
* Query optimization fundamentals
* SQL analytical querying
